# 平台跳躍遊戲 (Platform Game)

## 檔案說明

- `platform-game.html` - 遊戲主檔案，包含完整的平台跳躍遊戲

## 遊戲功能

| 功能 | 說明 |
|------|------|
| 玩家控制 | 使用 ← → 鍵移動，↑ 鍵跳躍 |
| 生命系統 | 3條生命 |
| 金幣收集 | 收集金幣獲得分數（每個10分） |
| 敵人系統 | 敵人來回移動，碰到會死亡 |
| 關卡系統 | 3個不同難度的關卡 |
| 勝利/失敗 | 通關或失敗時顯示相應畫面 |

## 遊戲操作

1. 點擊「開始遊戲」按鈕
2. 使用方向鍵控制角色移動和跳躍
3. 收集金幣、躲避敵人、到達終點
4. 通過所有3關即可通關
